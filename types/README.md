# @aikirun/types

Core type definitions for the Aiki durable execution platform. Typically not used directly — other Aiki packages import from it.

## Installation

```bash
npm install @aikirun/types
```

## Usage

```typescript
import type { WorkflowOptions } from "@aikirun/types/workflow-run";
import type { TriggerStrategy } from "@aikirun/types/trigger";
```

## Documentation

See the [Aiki documentation](https://aiki.run/docs).
